    Hand 的處理邏輯

    Hand 是一個使用 Github 來進行收發作業的框架。 這個框架之下的特色有:

    1. 學生的作業預設是 private，並且被所有組織所擁有
    2. 可以在發布作業之後快速針對學生的作業提供補丁(用於更新敘述)

    使用場景 1. 讓一群學生能夠使用 Github 來收發作業
